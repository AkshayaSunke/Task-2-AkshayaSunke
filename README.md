# DecodeLabs-Internship

## Expense Tracker 

This repository contains a simple console-based Expense Tracker application developed using Python as part of the DecodeLabs Internship Program.

The application allows users to:
- Add expenses
- View total spent amount
- Exit the application

This project demonstrates basic Python programming concepts and backend logic implementation.
